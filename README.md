# João Brasil | Transparência e Atuação Legislativa

Este projeto é um portfólio digital moderno voltado para a atuação pública de **João Brasil**. A plataforma foi desenvolvida para consolidar dados legislativos, métricas de presença e projetos de lei, oferecendo uma interface transparente e acessível para o cidadão.

![Banner do Projeto](https://images.unsplash.com/photo-1529107386315-e1a2ed48a620?auto=format&fit=crop&q=80&w=1200&h=400)

## 🚀 Tecnologias Utilizadas

- **React e TypeScript**: Para uma interface robusta e tipagem segura.
- **Vite**: Para um ambiente de desenvolvimento ultra-rápido.
- **Tailwind CSS**: Estilização moderna e customizada com foco em estética premium.
- **Motion**: Animações fluidas e micro-interações para uma melhor experiência do usuário.
- **Lucide React**: Biblioteca de ícones elegantes.

## ✨ Principais Funcionalidades

- **Painel de Métricas**: Visualização auditável de projetos apresentados, aprovados e presença em sessões.
- **Linha do Tempo de Atuação**: Detalhes sobre a trajetória e experiência na vida pública.
- **Portal de Projetos**: Lista detalhada de iniciativas legislativas com status em tempo real.
- **Transparência de Dados**: Integração visual com dados simulados de APIs oficiais (Câmara, Senado e TSE).

## 🛠️ Como Rodar Localmente

Certifique-se de ter o **Node.js** instalado em sua máquina.

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Acesse `http://localhost:3000` no seu navegador.
